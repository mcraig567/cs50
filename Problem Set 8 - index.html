This is a page all about Age of Empires II

AoE is a game created long ago that has had a recent surge in popularity. The goal is to conquer your enemy by either defeating them in battle, building a wonder, or collecting all the relics

Players start in the Dark Age, and work to advance to other Ages throughout the game. There are four ages in total: Dark Age, Feudal Age, Castle Age, and Imperial Age. Additional information about each age can be found using the navigation tab
